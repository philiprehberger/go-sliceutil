# Changelog

## 0.2.1

- Standardize README to 3-badge format with emoji Support section
- Update CI checkout action to v5 for Node.js 24 compatibility
- Add GitHub issue templates, dependabot config, and PR template

## 0.2.0

- Add `Find` — return first element matching predicate
- Add `FindIndex` — return index of first match, -1 if none
- Add `Any` — true if any element matches predicate
- Add `All` — true if all elements match predicate
- Add `SortBy` — sort by extracted key
- Add `Take` — return first n elements
- Add `Drop` — skip first n elements
- Add `Compact` — remove zero-value elements
